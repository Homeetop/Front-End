    Welcome to F&VT

    F&VT meaning Food and Vegetables Translator is a class project in CPE 510; Natural Language Processing (NLP) a 3 unit course, develop by Group 7 member. it is one of the requirements in passing the course. it a web app that takes the name of any fruit or vegetables and translate it to any nigeria language of choice.

    Go

    Guide

    Using F&VT is very easy, to get started all you need to do is to click go and it will take you to a section where you can search for any fruits or vegetables of yours for translation to any nigeria language of your choice. for now there are only three common nigeria language
